All the raw vhdl code used to implement Piccolo 80 bits.

There is 2 Top Module registered base and 2 top piccolo module. 1 register based and 1 classical top module for each implementation (Speed and ressources).

Some of this file are shared between the 2 implmentations : s_box.vhd , m_box.vhd , f_box.vhd and Round_permutation.
